**客群特征**: 商业地产（商场/购物中心/写字楼）运营企业，AI 应用处于"应用期向规模化期过渡"阶段。头部企业（华润万象生活、龙湖智创生活、凯德）已落地商场 AI 质检、合同智能审查、企业知识库等场景，AI 消耗占总云消耗约 3-8%，增长空间显著。IT 系统大量依赖外包团队开发，外包团队规模通常超过内部研发团队。

**关键需求**: 已落地 AI 场景规模化复制（AI 质检/合同审查/知识库）、AI 新场景拓展（智能客服/精准营销/数据分析助手）、外包团队 AI 提效（Qoder）、TOKEN 消耗量提升（MaaS 收入增长）

### 行业特征

- **AI 从"试点验证"向"规模化复制"过渡**：头部企业已落地 2-3 个 AI 场景，正推广至多商场/多项目
- **大模型 API 调用是 AI 消耗主力**：百炼大模型平台消耗占 AI 总消耗 40-60%，合同审查和知识库问答为核心场景
- **视觉 AI 从安防向运营延伸**：新兴趋势——客流统计、热力图分析、商铺布局优化，增长快于传统安防场景
- **IT 系统大量依赖外包开发**：物业系统、租户系统、营销系统等多为外包，AI 场景落地需要外包团队具备 AI 集成能力
- **CEO/CTO 层面对 AI 形成战略共识**：2025-2026 年多家头部企业高管公开强调 AI 数字化是核心竞争力

| 阶段 | 目标月消耗 | 关键举措 |

## 推荐架构方案

### 方案一：API 调用为主（当前阶段推荐）

```
商业地产系统 → 百炼大模型（qwen3.6-plus API）→ 合同审查/知识库/质检报告
             → 视觉智能开放平台 API → 商场 AI 质检/客流统计
             → 智能语音交互 API → 语音客服/导航
             → 智能客服 API → 租户/会员服务
```

**适用**：当前阶段全部商业地产客户  
**核心价值**：零 GPU 投入，按量付费，灵活扩展

### 方案二：GPU 自建（未来 1-3 年评估）

**触发条件**：AI API 月调用量超过 50 万次/日，API 成本超过自建 GPU 成本  
**推荐型号**：gn7i (A10) 3-5 台  
**月消耗**：约 3-10 万  
**应用场景**：自建大规模视频 AI 分析平台、行业专属模型微调

> ⚠️ 当前商业地产行业头部企业均采用 API 调用模式，暂无自建 GPU 大规模部署的公开案例，此方案待验证。

## 销售策略

### 关键决策人
- **CTO/CIO/数字化负责人**：AI 场景落地和技术方案决策
- **外包项目管理负责人**：Qoder AI 编程工具推广
- **CEO/总裁**：AI 数字化战略共识已形成，可引用行业高管公开表态作为切入点

### 切入时机
- 现有客户云资源续约/扩容期（AI 消耗提升窗口）
- 地产 IT 系统升级/外包项目启动期（Qoder 提效推广窗口）
- 新商场开业/存量商场改造期（AI 质检、智能客服部署窗口）

### 销售话术参考
