# 智谱 GLM 系列模型

> 最后更新: 2026-05-28
> 所属厂商: 智谱 AI（Zhipu AI）
> 产品类别: MaaS
> 状态: Published

**定位**: 智谱自研 GLM 系列大模型，清华 KEG 实验室背景，强调中国本土化、合规部署与 ARC（Agent/Reasoning/Coding）能力体系
**当前主推**: GLM-5.1（2026.03.27）
**适用**: 企业合规部署、政企客户、本地化推理、中文编程
**不适用**: 需要极致性价比的开源场景、对英文基准测试要求极高场景

## 核心能力与限制

### 核心能力

| 能力 | 说明 |
|------|------|
| **编程 SOTA** | GLM-5 SWE-bench 达开源 SOTA，比肩 Claude Opus 4.5 |
| **本地化部署** | 2025 年本地化部署收入 5.34 亿元，同比 +102.3%，是核心商业模式 |
| **政企合规** | 央企、政务、金融、能源等高合规要求行业渗透率高 |
| **中文优化** | 清华大学 KEG 实验室背景，中文理解能力强 |

### 核心限制

| 限制项 | 具体值 | 说明 |
|--------|--------|------|
| 开源生态 | 相对 DeepSeek/Qwen 开源力度较弱 | GLM-4 之前版本开源，新版本商业化为主 |
| 国际化 | 主要面向中国市场 | 海外竞争力相对弱 |
| 性价比 | 非极致 | 定位企业市场，非价格战路线 |

## 适用场景

### ✅ 适用

| 场景 | 推荐模型 | 说明 |
|------|----------|------|
| 企业合规部署 | GLM-4-Plus / GLM-5 | 本地化部署是核心商业模式 |
| 政企客户 | GLM-4 全系列 | 高合规要求行业首选 |
| 中文编程 | GLM-5 | SWE-bench 开源 SOTA |
| 中等推理任务 | GLM-4-Air / Flash | 性价比分层 |

### ❌ 不适用

| 场景 | 原因 |
|------|------|
| 极致性价比开源 | DeepSeek V 系列更有优势 |
| 英文为主的开源研究 | GLM 系列中文优化为主 |
| 全球市场 | 国际化能力相对弱 |
