**定位**: DeepSeek 推理专项模型系列，以纯强化学习（RL）为核心技术路线，专注复杂推理能力

## 核心能力与限制

### 核心能力

| 能力 | 说明 |
|------|------|
| **复杂推理** | GRPO 算法驱动，自主生成高质量思维链 |
| **数学证明** | 数学竞赛级问题解决 |
| **代码生成** | LeetCode、Codeforces 等竞赛题解决 |
| **幻觉控制** | R1-0528 幻觉率降低 45-50% |
| **开源 MIT** | 完全开源，可自托管部署 |

### 核心限制

| 限制项 | 具体值 | 说明 |
|--------|--------|------|
| 推理延迟 | 高 | 思考模式带来额外延迟 |
| 回复长度 | 长 | 思维链展开占用大量 token |
| 国内访问 | 需翻墙 | 需通过百炼国际站调用 |

## 适用场景

### ✅ 适用

|------|----------|------|
