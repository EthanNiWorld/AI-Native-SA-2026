## 一、公司概况

智谱 AI（全称：北京智谱华章科技股份有限公司，英文品牌 Z.ai / Zhipu AI）是一家从清华大学计算机系知识工程实验室孵化出来的通用人工智能公司，专注于自研 GLM 系列大模型，并通过 MaaS API 平台、AutoGLM 智能体、本地化部署、智谱清言（C 端）等业务实现商业化。是国内 **大模型六小龙** 中第一家完成 IPO 的公司[reference:0][reference:1]。

### 基本信息

**成立时间**：2019 年 6 月  
**总部**：中国北京（海淀区清华科技园）  
**性质**：港股主板上市公司（02513.HK，2026 年 1 月 8 日上市）  
**英文品牌**：Z.ai / Zhipu AI  
**控股股东**：清华系自然人持股结构 + 多轮机构投资人  
**现任 CEO**：张鹏（Zhang Peng）

## 五、关键产品矩阵

### 5.1 GLM 系列基础模型（核心产品）

**语言 / 推理 / 编程模型**

| 产品 | 发布时间 | 核心特点 |
|------|---------|---------|
| **GLM-130B（开源）** | 2022.08 | 千亿参数中英双语开源大模型，国产大模型早期标志 |
| **ChatGLM-6B（开源）** | 2023.03 | 国内最早一批开源中文对话模型，奠定开源生态基础 |
| **ChatGLM2 / ChatGLM3（开源）** | 2023.06 / 2023.10 | 持续迭代，单卡可部署 |
| **GLM-4** | 2024.01 | 首个全面对标 GPT-4 的国产模型，闭源旗舰 |
| **GLM-4-Plus / Air / Flash** | 2024 年 | 分层模型矩阵（旗舰 / 性价比 / 极速）|
| **GLM-4.5** | 2025 年 | ARC（Agent / Reasoning / Coding）三大能力体系建立 |
| **GLM-4.6** | 2025 年下半年 | 持续增强 ARC，上下文与函数调用大幅升级 |
| **GLM-5** | **2026.02.11** | 新一代旗舰，**SWE-bench Verified、Terminal Bench 2.0 等智能体编程榜单达开源 SOTA，比肩 Claude Opus 4.5**[reference:10][reference:11] |

**多模态模型（Cog 系列 + GLM-V）**

| 产品 | 发布时间 | 核心特点 |
|------|---------|---------|
| **CogView 系列** | 2021–2024 | 文生图模型，KEG 实验室代表作 |
| **CogVLM / CogAgent** | 2023–2024 | 视觉语言模型，CogAgent 专攻 GUI 操作 |
| **CogVideoX（开源）** | 2024.08 | 视频生成开源模型，对标早期 Sora |
| **CodeGeeX 系列** | 2022 起 | 代码模型 + IDE 插件，覆盖 VS Code / JetBrains |
| **GLM-4.6V** | 2025.12.08 | 视觉推理模型，原生多模态[reference:9] |

**GLM-5 设计理念**：从 "Vibe Coding" 升级为 **"Agentic Engineering"**，旨在应对复杂系统工程与长周期智能体任务，认为规模化仍是提升 AGI 智能效率的最佳路径[reference:11][reference:12]。

### 5.2 AutoGLM 与 AutoClaw（澳龙）— 智能体产品线

- **AutoGLM**：智谱自研智能体，具备 **浏览器自动化操作能力**（点击、表单、跨页面任务），是国内最早投入商用的"操作型 Agent"之一[reference:13]
- **AutoGLM Rumination（沉思）**：长周期任务智能体，可执行数小时级别的研究、报告生成
- **AutoClaw（澳龙）**：**国内首款一键安装的本地 OpenClaw 客户端**，内置 50+ Skills，集成 AutoGLM 浏览器操作能力[reference:13]
- **CodeGeeX Plus**：IDE 编程助手 + Agentic 编程模式（基于 GLM-5）

### 5.3 智谱清言（ChatGLM 应用）— C 端产品

- 类 ChatGPT 中文对话产品，免费 + 会员订阅模式
- 集成图像生成、文档分析、代码助手、AutoGLM 智能体等能力

### 5.4 MaaS 开放平台（bigmodel.cn）— 企业 / 开发者

- 面向企业与开发者的模型即服务平台
- **MaaS API ARR 截至 2026 年 3 月达 17 亿元，过去 12 个月增长 60×**[reference:14]
- **MaaS 平台毛利率 18.9%，同比近 5 倍提升**[reference:14]
- 提供"龙虾套餐"等团队订阅产品

### 5.5 本地化部署 / 一体机

- 面向央企、政务、金融的本地化大模型解决方案
- 提供模型授权、部署、调优、私有云一体化服务
- 2025 年收入 5.34 亿元，是公司最大收入板块[reference:8]

## 十一、智谱 AI 核心论文与影响力分析

### 核心研究脉络

智谱的研究主线为 **"知识工程 → 通用语言模型（GLM）→ Agentic Engineering"**：

- **知识工程根基**：清华 KEG 实验室在知识图谱、社会网络、学术挖掘领域有深厚积累，AMiner 学术搜索引擎是其代表作
- **GLM 架构创新**：GLM 提出 **自回归空白填充（Autoregressive Blank Infilling）** 训练目标，统一了 BERT 类与 GPT 类的预训练范式[reference:4]
- **Agentic 路径**：从 GLM-4 → 4.5 → 4.6 → 5，持续强化 ARC（Agent / Reasoning / Coding）三角能力，到 GLM-5 在编程智能体榜单达到开源 SOTA[reference:11]

### 标志性论文与模型

| 名称 | 时间 | 核心贡献 |
|------|------|----------|
| **GLM: General Language Model Pretraining with Autoregressive Blank Infilling** | 2022（ACL）| 提出 GLM 架构，统一双向理解与单向生成预训练目标，是 ChatGLM 系列的理论基石[reference:4] |
| **ChatGLM-6B（开源）** | 2023.03 | 国内最早一批中英双语开源大模型，单卡可部署，加速国产开源生态 |
| **GLM-130B（开源）** | 2022 | 千亿参数量级双语模型，国产大模型早期标志性成果 |
| **AgentBench / AgentTuning** | 2023–2024 | 智能体能力评测与训练方法论，奠定 AutoGLM 的研究基础 |
| **GLM-4.6V** | 2025.12 | 视觉推理多模态模型 |
| **GLM-5** | 2026.02 | 新一代旗舰，SWE-bench Verified、Terminal Bench 2.0 开源 SOTA，比肩 Claude Opus 4.5[reference:11] |

### 行业影响力

- **国产开源生态奠基者**：ChatGLM-6B 是国内开源生态最重要的早期项目之一，直接催生了大量上层应用
- **学术影响力**：清华 KEG 实验室在 ACL、NeurIPS、KDD 等顶会持续产出，唐杰教授本人为高被引学者
- **资本市场标杆**：2026.01.08 港股 IPO，**"全球大模型第一股"**，市值最高超 579 亿港元
- **政企渗透代表**：在央企、政务、金融的本地化部署上是国产大模型的旗帜性供应商
- **国际影响力**：GLM-5 在国际开源 Agentic 编程榜单达到 SOTA，标志中国开源大模型在 Agent 领域进入第一梯队
