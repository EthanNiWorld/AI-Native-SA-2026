## 是什么

Claude Code 是 Anthropic 推出的 AI 编程工具，专注于代码生成、理解和重构。

## 竞争格局（2026-04 更新）

### 编程领域竞对分析

| 维度 | Claude Code (Anthropic) | Codex (OpenAI) | Qoder (阿里云) |
|------|------------------------|----------------|----------------|
| 真实代码库应用 | **先发优势**：更早认识到价值 | 起步慢，但已追赶 | 企业级定位 |
| 编程竞赛指标 | 优秀 | **领先**：抽象指标最高分 | 良好 |
| 混乱代码库处理 | **领先**：早期投入 | 快速改进中 | 待验证 |
| 综合对比 | 多维度被 Codex 追平/超越 | 很多维度已领先 | 企业场景优势 |

### OpenAI 官方评价（2026-04）

> "在编程方面有一个例子，我们在'真实世界代码'上起步比 Anthropic 慢了。我们一直在编程竞赛的抽象指标上拿最高分，但把模型应用到混乱的真实代码库、真实世界数据上，这是 Anthropic 比我们早认识到的。但这也反过来提升了我们的执行力，现在 Codex 对比 Claude，我们在很多维度上已经领先了。"
> 
> —— Greg Brockman, OpenAI 联合创始人

**关键洞察**：
- Anthropic 的优势：更早理解真实世界代码的复杂性（混乱、不规范、遗留代码）
- OpenAI 的追赶：通过竞赛指标优势快速转化为实际产品能力
- 竞争促进了双方执行力提升
