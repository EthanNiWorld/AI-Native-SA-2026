# Wan2.6 Demo

> 创建日期: 2026-04-20
> 关联模型: wan.md

## Demo 1: Wan2.6 文生视频 (t2v)

```python
from dashscope import VideoGeneration

response = VideoGeneration.call(
    model="wan2.6-t2v",
    prompt="一只金色柴犬在樱花树下奔跑，阳光透过花瓣洒落，电影质感，慢动作",
    size="1280*720",
    duration=5
)

print(response.output.video_url)
```

## Demo 2: Wan2.6 图生视频 (i2v)

```python
from dashscope import VideoGeneration

response = VideoGeneration.call(
    model="wan2.6-i2v",
    image_url="https://example.com/product.jpg",
    prompt="产品缓缓旋转，背景虚化，光影流动",
    size="1280*720",
    duration=5
)

print(response.output.video_url)
```

## Demo 3: Wan2.6 图像生成 (Image)

```python
from dashscope import ImageGeneration

response = ImageGeneration.call(
    model="wan2.6-image",
    prompt="赛博朋克风格的上海外滩夜景，霓虹灯倒映在江面上，8K超高清",
    size="1024*1024",
    n=1
)

print(response.output.results[0].url)
```

## 备注

- 文生视频支持中文提示词，描述越详细画面越可控
- 图生视频适合电商产品动态展示
- 万相平台提供可视化创作界面，无需代码
